# Kuaför Randevu Sistemi

Bu proje, kuaför salonları için online randevu sistemi sağlar.

## Özellikler

- Kullanıcı ve kuaför kayıt sistemi
- Kuaför arama ve filtreleme
- Randevu oluşturma ve yönetme
- Kullanıcı profil yönetimi
- Kuaför profil ve hizmet yönetimi

## Teknolojiler

- Next.js 14
- TypeScript
- Firebase (Authentication & Firestore)
- Tailwind CSS
- React Icons

## Kurulum

1. Repoyu klonlayın
```bash
git clone https://github.com/kullanıcı-adınız/kuafor-randevu.git
cd kuafor-randevu
```

2. Bağımlılıkları yükleyin
```bash
npm install
```

3. Çevre değişkenlerini ayarlayın
- `.env.example` dosyasını `.env` olarak kopyalayın
- Firebase yapılandırma bilgilerinizi ekleyin

4. Geliştirme sunucusunu başlatın
```bash
npm run dev
```

5. Tarayıcıda açın
